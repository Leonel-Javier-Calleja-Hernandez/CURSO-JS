¿Por qué NO se debe usar innerHTML con JAVASCRIPT?

si utilizamos el innertHTML para añadir o hacer un contenido dinamico dentro de un caja o div este da error a los eventos deje de funcionar o puede que solo funcione uno de todos los botones,

cuando tenemos un div y dentro de este hay algun boton o elemento que se le puede dar click para hacer alguna funtion etc.. al unicio puede funcionar pero despues falla ,al utilizar el innerHTML Los elementos clicables de dicha "#caja" dejan de funcionar.

¿POR QUE OCURRE ESTE ERROR?

por que al utilizar el operador de innertHTML+= el agrega el valor que nosotro le pongamos pero no soluciona los problemas que proboca

¿CUANDO UTILIZAR EL innertHTML?

Solo utilizarlo para eliminar el contenido que hay en una caja y cambiarlo por otro nuevo contenido pero nunca para insertar nuevo contenido elementos html y menos si que tenga eventos o funtion

Utilizar el insertAdjacentHTML

este funciona para cuando queremos insertar elementos que contengan eventos como onclikck y funciones

entre los parentesis se pone la ubicacion de donde lo queremos poner y el contenido a enviar

Soluciones

insertAdjacentHTML

cambiar el innertHTML por utilizar el insertAdjacentHTML(1 , 2)

Ejemplo: insertAdjacentHTML("beforeend",`elemento`);

el insertAdjacentHTML corrige lo que el innerHTML no hace.

1. donde lo queremos añadir

arriba de caja = beforebegin

queda antes fuera de la caja y arriba

abajo de caja = afterend

queda fuera de caja y abajo

antes del primer hijo de caja = afterbegin

dentro de la caja y de primeras antes de el hijo

despues de el ultimo hijo = beforeend

dentro de la caja y despues de el ultimo hijo

2.el elemento a incresar en el sea div,buttun,h1,etc..

el innerHTML y el insertAdjacentHTML dentro de el "for"

cuando estemos utilizando el innerHTML dentro de un for, si queremos que este tenga un evento como ejemplo de click. este evento onclick se debe hacer en otro for o fuera a diferencia de el insertAdjacentHTML que este si se puede poner los eventos dentro de el for mismo.
