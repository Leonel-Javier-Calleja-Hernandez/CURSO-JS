LAS VARIABLES

Es un palabra o un conjunto de letras que sirven para guarda un valor(un dato)

NOMBRES DE UNA VARIABLE

el nombre tiene que empezar por letras MAYUSCULA o minuscula "_" o $
ejemplos= $HOla, _hol4, hol_4, Hola

TIPOS
var, let, const

var = en las funtion al crear una varible dentreo de {} solo tiene valides en las {}

let = al momento de definir let dentro de las llaves{} o bloque en el que se encuentra solo tiene valor hay y por ende su valor se elimina la salir de {}

const = se nombran con MAYUSCULAS y no es posible cambier el valor por que sino da error

*son para Guardar datos
*nunca empiezan por un numero siempre por una letra
*nunca tienen espacios
*pero si es muy larga se pueden iniciar la otra palabra con MAYUSCULA O guion bajo"_" para dar espacios

ejemplo: var nombreDeVariable = "valor de vari4ble"

VALORES DE ERROR

not defined

lo que se esta llamando no esta definida y puede ser por que aun no esta creada
ejemplo: mostrar una varible en consola no creada aun

undefined

se produse cuando creamos una variable y no tiene ningun valor
ejemplo: llamar una variable que no tenga ningun valor

null

no tiene valor y esta vacio
ejemplo: mostrar una varible en console.log que no tenga nada

NaN

se interpretado como imposible
ejemplo: multiplicar un numero y un texto
