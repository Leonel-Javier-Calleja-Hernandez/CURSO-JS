LAS VARIABLES

Es un palabra o un conjunto de letras que sirven para guarda un valor(un dato)

NOMBRES DE UNA VARIABLE

el nombre tiene que empezar por letras "_" o $
ejemplos= $HOla, _hol4, hol_4, Hola

TIPOS =var, let, const

var = en las funtion al crear una varible dentreo de {} solo tiene valides en las {}

let = en los condicionales("if,else if,else") y en los bucles los let solo tiene tiene valor dentro de las llaves{}

const = se nombran con MAYUSCULAS y no es posible cambier el valor por que sino da error

*son para Guardar datos
*nunca empiezan por un numero siempre por una letra
*nunca tienen espacios
*pero si es mu larga se pueden iniciar la otra palabra con MAYUSCULA O guion bajo"_" para dar espacios
