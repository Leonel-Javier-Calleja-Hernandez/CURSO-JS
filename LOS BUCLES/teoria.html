LOS BUCLES

la repeticion de una serie de acciones al instante tantas veces como sea necesario.fundamentales para la creación de cualquier web que aporte contenido dinámico (o no). Junto con los arrays y las variables tratadas en los vídeos anteriores en este lo complementamos con los bucles. Veremos qué es un bucle, de qué se compone, qué código utilizamos para la creación de bucles, tipos de bucles (for y while) en JavaScript.

TIPOS DE BUCLESs

for

en este sele tiene que indicar 3 datos
1.una variable para que sepa por donde tiene que empezar a contar.
2.hasta cuando debe de contar.
3.el incremento"i++" o decremento"i--" cada vez que pase por el for.

Para qué sirve la variable contador del FOR

es para saber el numero que tiene, y este tambie funciona para utilizarlo dentro de el bucle

while

es una variante de los bucles, pero se caracteriza por tener un condicional. y al finalizar dicha condicion continuaria con las lineas siguientes
